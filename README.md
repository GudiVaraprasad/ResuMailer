# 📧 ResuMailer

**ResuMailer** is an AI-powered Streamlit application that crafts personalized cold emails by analyzing your resume and targeted job descriptions. Enhance your job application process with smarter, tailored outreach. 🚀

## ✨ Features

- **AI-Generated Emails**: Utilizes the Groq AI open-source LLM API (gemma2-9b-it) to draft customized emails based on your resume and job descriptions. 🤖
- **User-Friendly Interface**: Built with Streamlit for an intuitive and interactive user experience. 🖥️
- **Efficient Job Applications**: Streamlines the process of reaching out to potential employers with personalized content. 📈

## 🛠️ Installation

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/GudiVaraprasad/ResuMailer.git
   cd ResuMailer
   ```

2. Update your information in sender_details.json

3. **Set Up a Virtual Environment**:
   - *Windows*:
     ```bash
     python -m venv venv
     .\venv\Scripts\activate
     ```
   - *macOS/Linux*:
     ```bash
     python3 -m venv venv
     source venv/bin/activate
     ```

4. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

5. **Run the Application**:
   ```bash
   streamlit run app.py
   ```

## 📄 Usage

1. **Upload Your Resume**: Provide your resume in PDF/Word/Text format. 📄
2. **Input Job URL**: Enter the job URL. 📝
3. **Generate Email**: Click 'Generate Cold Email' to receive a personalized cold email tailored to the job. ✉️
